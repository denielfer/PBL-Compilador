# Analisador Léxico

## Execução do código
Para executar o código do problema basta utilizar o comando abaixo na pasta raíz:
`python main.py`
caso não funcione, tente:
`python3 main.py`

Este arquivo fará a leitura de todos os arquivos de testes presentes na pasta *files* e suas respectivas análise léxicas. Após isso, serão gerados arquivos de saídas para cada um deles e armazenado os logs de execução em um arquivo .txt na pasta raíz.
